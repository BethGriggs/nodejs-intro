Prerequisites

    • Install Node.js (Latest LTS- currently v8.9.3) from either:
    • https://nodejs.org/en/, download latest LTS, confirm that Node is in your path by typing `node --version` in your command line
    • or https://github.com/creationix/nvm#installation (Unix) or https://github.com/jasongin/nvs (Windows). These allow you to quickly swap between Node versions.
  • Use a text editor of your choice. We will not need to utilise the extra features some editors/IDEs provide in this begginers course.
